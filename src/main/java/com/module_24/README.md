# CV Evaluator API

A Spring Boot REST API that accepts a CV image and returns a structured evaluation score using Google Gemini AI Vision.

---

```

├── dto/
│   └── CVEvaluationResponse.java

```

## Setup & Run

### 1. Get a Gemini API Key

1. Go to [https://aistudio.google.com](https://aistudio.google.com)
2. Sign in with your Google account
3. Click **"Get API Key"** → **"Create API Key"**
4. Copy the key

### 2. Add the API Key

Open `src/main/resources/application.properties` and replace the placeholder:

```properties
gemini.api.key=YOUR_GEMINI_API_KEY_HERE
```

### 3. Run the Application

```bash
./mvnw spring-boot:run
```

The server starts on `http://localhost:8080`

---

## API Endpoint

### POST `/api/cv/evaluate`

Accepts a CV image file and returns an AI-generated evaluation.

| Property         | Value                                   |
|------------------|-----------------------------------------|
| Method           | `POST`                                  |
| URL              | `http://localhost:8080/api/cv/evaluate` |
| Body Type        | `form-data`                             |
| Parameter        | `file` (type: File)                     |
| Accepted formats | JPG, PNG                                |

---

## Postman Instructions

1. Open Postman
2. Create a new request
3. Set method to **POST**
4. Set URL to `http://localhost:8080/api/cv/evaluate`
5. Go to **Body** tab → select **form-data**
6. Add a key named `file`, change its type to **File** (hover over the key field to see the dropdown)
7. Select your CV image file
8. Click **Send**

---

## Sample Request & Response

### Request

```
POST http://localhost:8080/api/cv/evaluate
Content-Type: multipart/form-data

file: [my_cv.jpg]
```

**Postman setup:**

```
Method : POST
URL    : http://localhost:8080/api/cv/evaluate
Body   : form-data
  Key  : file    (Type: File)
  Value: my_cv.jpg
```

## Notes

- Only image files are supported (JPG, PNG)
- Maximum file size: 10MB
- The AI evaluates only what is **visible** in the image — no assumptions are made
- Response is always in JSON format
